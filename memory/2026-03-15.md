## Update: /top Command Added

### 5. Top Holders Command for himmel.py
- Added `/top <token_address>` command
- Fetches top 10 holders of any Solana token
- Uses Birdeye API as primary source
- Helius API as optional fallback
- Display features:
  - Ranked list (1-10)
  - Wallet address (truncated)
  - Percentage held with color indicators:
    - 🔴 >= 10% (Whale)
    - 🟠 >= 5% (Large holder)
    - 🟡 >= 2% (Medium holder)
    - 🟢 < 2% (Small holder)
  - Token amount
  - Summary: Total % controlled by top 10

### API Endpoints Used
- Birdeye: `GET /public/token_holders?address={token}&limit=10`
- Helius: `GET /v0/tokens/{token}/holders?limit=10` (fallback)

### Required Environment Variables
```
BIRDEYE_API_KEY=your_key_here
HELIUS_API_KEY=your_key_here  # Optional
```
