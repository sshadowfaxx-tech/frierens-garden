**Fix:** Changed SQL to calculate winrate from `winning_trades / total_trades`:
```sql
CASE 
    WHEN p.total_trades > 0 THEN (p.winning_trades::float / p.total_trades * 100)
    ELSE 0 
END as wallet_winrate
```
