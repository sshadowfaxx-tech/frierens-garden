## Hardware Decision - Home Server

### User's Available Hardware
- **CPU**: AMD FX 6300 Six-Core 3.5 GHz
- **RAM**: 8GB
- **Storage**: 1TB SSD

### Assessment
- **Grade**: Excellent for home server deployment
- **Capacity**: 10x overkill on CPU, 4x on RAM, 100x on storage vs minimum
- **Can handle**: 100+ wallets, web dashboard, Grafana, multiple bots
- **Architecture**: x86_64 (perfect Docker compatibility)

### Comparison to Alternatives
| Spec | FX 6300 | Raspberry Pi 5 | Hetzner CX21 |
|------|---------|----------------|--------------|
| CPU | 6 cores | 4 cores | 2 vCPU |
| RAM | 8GB | 8GB | 4GB |
| Storage | 1TB SSD | 256GB max | 40GB |
| Power | ~95W | ~15W | N/A |
| Upfront | $0 | ~$195 | $0 |
| Monthly | ~$8-12 elec | ~$3 elec | ~$5.80 |

### Decision
**RECOMMENDED**: Use the FX 6300 for home deployment
- Zero upfront cost (already owned)
- Massive headroom for scaling
- x86 compatibility (no ARM issues)
- 3-year TCO ~$300-400 (still competitive)
- Break-even vs new Pi 5: ~18 months
