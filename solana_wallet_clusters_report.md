## METHODOLOGY FOR DETECTING CLUSTERS

### Key Indicators:

1. **Funding Pattern Analysis**
   - Wallets funded from same source within short time windows
   - Fresh wallets (<7 days old) with large inbound transfers
   - Identical funding amounts across multiple wallets

2. **Timing Coordination**
   - Multiple wallets buying same token within same block/second
   - Pre-announcement funding (hours before public knowledge)
   - Coordinated selling patterns (cascading exits)

3. **Behavioral Signatures**
   - Same tokens traded across wallet group
   - Similar position sizing relative to wallet balance
   - Cross-wallet fund movements (circular transactions)

4. **Profit Patterns**
   - Consistent profitability across entire cluster
   - Unrealistic win rates (>80% on new launches)
   - Large realized gains within hours of entry

### Tools for Detection:
- **Bubblemaps:** Visual wallet clustering and connection mapping
- **Nansen:** Smart money labeling and wallet profiling
- **Lookonchain:** Real-time transaction monitoring
- **Arkham Intelligence:** Entity identification and tracking
- **Dune Analytics:** Custom queries for pattern detection
